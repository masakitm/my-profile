# 概要
厳しくて貧乏な家に育った、野良猫で岩男で助けたい症候群の自立武闘派男性。  
恋愛は続かず仕事もつらい。東京暮らしも合わない。
2018年からアドラー心理学などを読み出す。  
2021年根本裕幸氏のブログと執着本がぶっ刺さる。  
  
手放しワークなどを通して、「実母にすら愛されなかった自分は誰にも愛されてはいけない」「僕も僕自身を愛してはいけない」と無意識に決めていたことに気づく。  
自分をとんでもなく抑圧していたことに気づく。
本当に幸せな生き方を探し始める。  

# 家族構成
母 ヒステリック。精神が不安定。理不尽に叱る。厳しい。一方的。
父 DVを受けて育った。理不尽に叩く。口下手。一方的。
妹 中卒。引きこもり。ニート。

# 幼少期
* 厳しくて貧乏な家に育った。
* 祖父が父にDVを振るう人だった。  
* 当初は同居していてケンカばかりだった。  
* 1−2歳の僕は「ケンカやめてよー！」とよく泣いて仲裁に入ったらしい。  
* 3-4歳のとき祖父の借金がきっかけで夜逃げ同然で父・母・妹とともに母の実家へ逃げる。  
* その後4歳で団地へ入居する。
* おぼえたての言葉「ストレス」を使ってみたくて、
  元気のなかった母に「それってストレスってやつ？ねえねえ」と何度も話しかけたら
  「うるさい！！！！」と頭ごなしに叱りつけられた
# 小学校
* 母との癒着のはじまり。
* 父の一方的な意見の押しつけ。
* ゲーム、特にスーパーファミコンが大好きなのに「目が悪くなるから」と買ってもらえない。
* 激しく怒ったり優しくなったり精神に不安定さのある母。
* 家にいない父。
* 「ゲームも買ってもらえない貧乏人のくせに！」「ゲームばっかしにうちに来ないでくれる！？」と友達に罵られる日々。
* 学校では勉強ができて人望は多少あったらしい。
* 運動は苦手で強いコンプレックスを抱く。
* 「お前は鼻の穴がでかい」「お前は汗臭い」「お前の生まれたときの顔はアル中の祖父そっくりでドン引きした」と母に言われる。
* 98点を取ると「なんで100点じゃないの？」と母に詰められることが続き完璧主義になる。
* 見た目に強いコンプレックスを抱く。
* お祭りで自分の大好きなお菓子を初めて自分で手に入れて感動。取っておいたものを父に全部たべられ、「賞味期限が切れるから」と逆ギレされる。
* 「普通のサラリーマンはつまらん。お前はものを作る人間になれ」と強く父に教えられ、普通の生き方をしてはいけないと思うようになる。
* 詳しい事情は知らないが妹が不登校になる。
* 「〜ちゃん好きかも」と言ったら同級生にめちゃくちゃからかわれ、自分が誰かを好きになってはいけないと感じ始める。
* 初めてほしいと思ってから8年後、スーパーファミコンを買ってもらう。4歳から12歳になっていた。
  このときは信じられず、喜びも悲しみしなかった。ただ涙が溢れた。夢だと思った。1週間は起きるたびまずスーファミを見に行った。
  それまでクリスマスプレゼントは欲しくもなんともない教育玩具や工具などばかりだったから。
  両親は僕が何をほしいかに興味がなかったんだろう。
* 小4で転校してきた少年に嫌がらせをされるようになる。理由はよく知らないが、目立ってて気に入らなかったのではないかと思われる。
* 母に「〜してほしい」というと「お母さんだってそんなことされたことなかったわね！！！」とヒステリーを起こされた。


# 小学校 - 転校

* 引越し先の街が全く僕に合わず苦労する。
* 裕福でない家なのに大きなローンを背負う。
* 引っ越し先に全く馴染めずいじめられる。
* 小学6年生での転校というのもタイミングが悪かった。関係性が出来上がっていて入れない。
* 担任の先生が「マサキくんが一人で本ばっか読んでるのはお前らが遊んでやらないからだろ！！！」とクラスメイトにブチギレ。
  「僕のために怒ってくれる人が人生で初めていてくれた」と感動し泣く。
* 転校して友達ができなくてつらいと母に相談すると「お前がそんなふうにウジウジしてるからだろ！！！」とキレられ、自分に頼れる人間はいないと知る。
* 妹がまた不登校になる。

# 中学校
* 父の会社が倒産する。
* ローンが払えず母がパートに出る。
* 母の精神状態がますます悪化。パートの前に嫌すぎて吐いてから出勤する、子どもの前で「金がない金がない金がない…」と延々つぶやくなど。
* 妹は引きこもりになり無言でキレ続ける暴君と化す。
* 父は母とケンカするか非正規で働くか遅くまで家にいないかのどれか。
* 「自分は強くならねばならない」と深く決心する。一人称を「僕」から「俺」に変える。
* 奇人変人ポジションが確立。男子からも女子からもいじりといじめの中間の嫌がらせを受け続ける。
* いじられつつも味方のいない中1人で抵抗しつづける。
* ある日たまたまいじめ側の一人と仲良くなる。それをきっかけに「なんか面白いやつ」と思われいじめっ子たちとツルむようになる。
* 同級生の対して好きでもないけどなんか人気があった子に告って振られ、次の日から全女子にシカトされる。卒業まで続いた。
  ここで自分は気持ち悪くてモテない人間なんだと深く心に刻まれる。

# 高校
* 人生で一番つらい時期。
* 母のメンタルヘルスも妹の引きこもりもますます深刻化。家に居場所がない。
* 中学時代の親友のいない進学校へ進学。行きたい高校を選びたかった。でも選べなかった。結果周囲にながされ進学校へ行くしかなかった。
* 1年1学期に頑張って女子に話しかけまくり、なんか面白いやつとしてたくさんの女子と仲良くなる。
* 男の友達はできず。
* 2学期になると急に女子にもシカトされるようになる。理由は知らない。
* この頃から自慰行為の回数が異常になっていく。それ以外に心を癒すものが何もなかった。辛かった。
* 自傷のかわりに自慰をした。痛くても何も出なくてもした。自分を罰するかのようだった。
* 自分も引きこもりになりたいくらいつらいのに、妹がいるから「自分だけはマトモでなくてはいけない」と強く思い込み通う。
  「カネがない」を連呼する母の前で「転校したい」とは言えなかった。
* 不眠症になる。夜中3時になっても眠れず、朝7時には「0時限目」と称して授業が始まる。不眠の頭で授業についていけなくなる。
* 告って振られて総シカトをまた経験する。学校にも居場所がなくなった。自尊心はもう完全にボロボロになった。
* 自分は穢れているんだ。日本人女性は僕なんかよりイケてる男たちが相手するものなんだ。父や母すら自分を愛さない。自分には居場所がない。自分は生きていてはいけない。そう本気で信じ込むようになった。
* 毎日最低5回は自殺を考えるようになる。
  包丁を見ては自分に刺す妄想をする。
  車を見ては河川を見ては飛び込み自殺を計画する。
  ついでにクラスメイトと学校の全員を大量虐殺して自分も死ぬ妄想をする。
  軽く2000回は自殺を考えたと思われる。
* 自分なんかが日本人女性に愛されることは決してないんだと本気で信じ込んでいた。
* 日本人女性に性的興奮できなくなっていることに気づく。また、女性の前で安心するという感覚が全くなくなっていることに気づく。
* 金がない実家に金を出させるわけにはいかない、と思いつめて虫歯を放置する。歯を四本失う。
* 「こんな地元絶対に離れて東京へ行くんだ」と決め、方言を矯正する。

* 自殺願望は1日1回になる。
* 「自分には価値がないんだから付加価値をつけねばならない」と思い込み作曲を始める。
* 高校よりはマシ。
* 授業中に海外エロサイトを見すぎて友達なくした。
* 自傷行為みたいな荒々しく痛々しい自慰行為しかできなくなっていた。
* ネットで知り合った女性と付き合うが一週間で別れる。

# 20代前半 - 地元・大阪

* 極貧ゆえ100均で買った小麦粉を焼いて食べる。
  あまりのまずさに「おれはこんなまずいものを食べるために、こんなに辛い思いをしているのか」「おれの人生はこんなもののためにあるのか」と本気で泣く。
* バイトすらまともにこなせる精神状態ではなかった。
* 他人としゃべることが恐ろしすぎた。
* 父が脳の病気で倒れた。今後の心配と同時に「やっと父の圧から解放される」と喜びを感じた。
* 父の後釜として印刷デザインの仕事をする。

* 足の指が折れた朝、吸えないタバコを拾って火を付けずにくわえたままバイトに出勤する。
* 音楽は上達したが「お前はもっと売れる曲を作れ」「でないと意味がない」「もっとこうしろ」と過干渉を受け鬱がひどくなり作曲をやめる。
* やっと毒親から自由になった。
* まともに就職できる精神状態ではなかった。

* ネットで知り合った女性と何人か付き合うが一週間とか二週間で手も繋がず別れる。
* 逆ナンされたりホテルに誘われたり告られたりするが、親密感の恐れなどもあり全部逃げる。
